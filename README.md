# Mac Mini Serveur (MMS)

Pull de tous les submodules : `git submodule update --init --recursive`

Les différents projets déployés sur le MMS :
- [ADGuard](https://github.com/superbionicle/ADGuard-Home)

    - port 53:53/udp
    - port 784:784/udp
    - port 853:853/tcp
    - port 3000:3000/tcp
    - port 80:80/tcp
    - port 443:443/tcp
- [Teamspeak](https://github.com/superbionicle/Teamspeak)
    - port 9987:9987/udp
    - port 10011:10011
    - port 30033:30033
